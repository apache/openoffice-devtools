Crea un progetto di tipo Applicazione Client per Apache OpenOffice.
È una applicazione client standard J2SE con il supporto integrato alle API Apache OpenOffice. La funzione principale include un frammento di codice per eseguire il bootstrap UNO e ritornare un contesto iniziale per il componente. L'implemtnazione attuale avvia automaticamente Apache OpenOffice o si connette a una istanza attiva di Apache OpenOffice. La comunicazione avviene tramite una connessione di tipo 'named pipe'.
