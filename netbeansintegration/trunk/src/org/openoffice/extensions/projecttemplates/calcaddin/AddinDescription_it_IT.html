Crea un progetto Apache OpenOffice Calc Add-In.
Un Add-In per Calc È una specifica estensione di Apache OpenOffice che aggiunge nuove funzionalità  tramite Add-In per Calc integrati. Queste funzioni possono essere utilizzate direttamente dalla tradizionale interfaccia utente dedicata all'applicazione di gestione dei fogli di calcolo in Apache OpenOffice.
Una procedura guidata consente l'inserimento di informazioni relative alle funzioni dell'Add-In per Calc e crea un nuovo progetto esteso J2SE di libreria di classi, generando i relativi file UNO IDL e un nuovo file schema sorgente in Java per l'implementazione delle nuove funzionalità  dell'Add-In.
