Apache OpenOffice Calc アドインプロジェクトを作成します。
Calc アドインは、オフィス用に特化した拡張機能です。新しい Calc アドイン関数を組み込むことで Apache OpenOffice の機能を拡張します。これらの関数は、Apache OpenOffice 表計算アプリケーションの通常のユーザーインタフェースで、直接使用できます。
このウィザードは新しい Calc アドイン関数に必要な情報を収集し、新しい拡張 J2SE クラスライブラリ プロジェクトを作成します。また UNO IDL ファイルと、新しいアドイン関数を実装する Java スケルトン ソースファイルも生成します。
