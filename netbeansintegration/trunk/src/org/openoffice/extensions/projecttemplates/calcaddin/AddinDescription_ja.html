OpenOffice.org Calc アドインプロジェクトを作成する。
Calc アドインは、オフィス用に特化された拡張機能で、新しい組み込み Calc アドイン関数を使用して OpenOffice.org 機能を拡張します。これらの関数は、OpenOffice.org 表計算ドキュメントアプリケーションの通常のユーザーインタフェースで、直接使用できます。
ウィザードによって新しい Calc アドイン関数に関する情報が収集され、新しい拡張 J2SE クラスライブラリプロジェクトが作成されます。また、UNO IDL ファイルが生成されて、新しいアドイン関数を実装する Java スケルトンソースファイルが生成されます。
