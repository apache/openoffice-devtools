Commands can be inserted back into the menu structure if they have been deleted. If you have not deleted any commands, you cannot insert commands back into the menu structure.
Menu The command is inserted into the selected menu, or after the selected element if a command or separator is selected. If nothing is selected, the command is inserted at the end of the top-level menu.
Toolbar The command is inserted after the selected command or separator. If nothing is selected, the command is inserted at the end of the Toolbar.
