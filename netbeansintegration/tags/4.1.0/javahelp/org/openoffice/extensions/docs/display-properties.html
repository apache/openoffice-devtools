Working with Apache OpenOffice Extensions: Display Properties

See Also 

This panel shows mostly localized data that is displayed in the extension manager. Note that all the information gathered here will be stored in the description.xml file.

Current Locale All localized properties are displayed for the current locale. Locales that contain entries are displayed bold when the drop down list is opened.
Default Locale When no data is available for the current locale of the user in Apache OpenOffice, the entries for the default locale are chosen.
Display Name The name of the Extension.
Description Link to a file that contains a description of the Extension. With a click on the browse button, a file from the file system can be chosen.
The button labelled "..." opens the description for editing.
Publisher Name Name of the publisher.
Publisher Link Link to the publisher's web site.
License Link to a file that contains the license of the Extension. With a click on the browse button, a file from the file system can be chosen.
The button labelled "..." opens the license for editing.
License Accept By User Or Admin Determines who has to acknowledge the license.
If "User" is chosen, every user must read the license. The Extension has to be installed by every user - it is not possible that an administrator does this for all.
If "Admin" is chosen, the Extension can be registered by an administrator once and every user of Apache OpenOffice can work with the extension without getting notified about the license.
Default Icon A default icon for the Extension. With a click on the browse button, a file from the file system can be chosen.
The button labelled "..." opens the icon for viewing.
High Definition Icon The high definition icon for the Extension. With a click on the browse button, a file from the file system can be chosen.
The button labelled "..." opens the icon for viewing.
