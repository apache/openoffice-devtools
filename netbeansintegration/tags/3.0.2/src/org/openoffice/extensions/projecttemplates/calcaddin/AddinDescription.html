Creates a Apache OpenOffice Calc Add-In project.
Calc Add-Ins are specialized office extensions which extends Apache OpenOffice functionality with new built-in Calc Add-In functions. These functions can be used directly in the normal user interface of the Spreadsheet application in Apache OpenOffice.
A wizard collects information about new Calc Add-In functions and creates a new extended J2SE class library project, generates UNO IDL files and generates a Java skeleton source file implementing the new Add-In functions.
