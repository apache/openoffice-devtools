Crea un progetto Apache OpenOffice Add-On.
Un Apache OpenOffice Add-On è un componente UNO esterno che fornisce una o più funzionalità tramite l'interfaccia utente di Apache OpenOffice. Un tipico Add-On è disponibile come package UNO per semplificare la distribuzione con il tool pkgchk. In aggiunta al package UNO predefinito, un package Add-On contiene i file di configurazione contententi le specifiche per l'interfaccia utente, la registrazione di uno schema di protocollo e le istanze relative al primo utilizzo.
