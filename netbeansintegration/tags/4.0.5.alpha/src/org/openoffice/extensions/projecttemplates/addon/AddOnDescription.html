Creates a Apache OpenOffice Add-On project.
A Apache OpenOffice Add-On is an external UNO component providing one or more functions through the user interface of Apache OpenOffice. A typical Add-On is available as a UNO package for easier deployment with the pkgchk tool. In addition to an ordinary UNO package, an Add-On package contains configuration files which specify the user interface, registration for a protocol schema, and first-time instantiation.
