Cria um projeto de Aplicação cliente do OpenOffice.org.
É um projeto cliente J2SE padrão com suporte integrado a API do OpenOffice.org. A função principal contém um trecho de código para o bootstrap do UNO e retornar um contexto de componente inicial. A implementação corrente automaticamente inicializa o OpenOffice.org padrão ou conecta a uma instância em execução do OpenOffice.org. A comunicação é feita usando uma conexão pipe nomeada.
