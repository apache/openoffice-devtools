Cria um projeto Calc Add-In do OpenOffice.org.
Calc Add-Ins são extensões especializadas do office que estendem funcionalidades do OpenOffice.org com novas funções Calc Add-In embutidas. Estas funções podem ser utilizadas diretamente pela interface com o usuário normal da Planilha do OpenOffice.org.
Um assistente coleta informações sobre as novas funções do Calc Add-Ine e cria um novo projeto de biblioteca de classe J2SE extensível, gera os arquivos UNO IDL e gera um arquivo esqueleto fonte em Java implementando as novas funções Add-In.
