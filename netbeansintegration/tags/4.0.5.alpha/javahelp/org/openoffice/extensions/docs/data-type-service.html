Use services and multi-inheritance interfaces to specify complete objects. The objects can have many aspects.
Properties
  • Name: The name of the polymorphic struct.
  • Package: The package structure.
  • Interface: The interface belonging to this service. Note: This service can only be created when an interface is specified.
