Cria um projeto Add-On do OpenOffice.org.
Um Add-On do OpenOffice.org é um componente UNO externo que provê uma ou mais funções através da interface do usuário do OpenOffice.org. Um típico Add-On está disponível como um pacote UNO para facilmente ser implantado com a ferramenta pkgchk. A diferença para um pacote UNO comum é que um pacote Add-On contém arquivos de configuração que especificam a interface com o usuário, registram um esquema de protocolo, e a primeira instanciação.
