Description
-----------

O Apache OpenOffice® é o programa de produtividade gratuito e de código aberto da Apache Software Foundation.

O Apache OpenOffice é constituído por seis aplicações de produtividade pessoal: processador de texto (e a sua componente de autoria para a web), folha de cálculo, apresentações, desenho, editor de equações e base de dados. Para algumas funcionalidades é necessário ter instalado um Java Runtime Environment (JRE).

What's new in this version?
---------------------------

O Apache OpenOffice 4.1.11 é uma edição de Segurança que incorpora correcções de erros importantes. Aconselha-se a todos os utilizadores do Apache OpenOffice 4.1.10 ou anterior a proceder à actualização para esta versão.

Product features
----------------

Writer - um processador de texto que pode utilizar para qualquer assunto, desde uma carta rápida até a um livro completo.
Calc - uma poderosa folha de cálculo com todas as ferramentas que necessita para calcular, analisar e apresentar os seus dados em relatórios numéricos ou gráficos expressivos.
Impress - o modo mais rápido e mais poderoso de criar apresentações multimédia efectivas.
Draw - permite-lhe criar tudo desde simples diagramas até ilustrações 3D dinâmicas.
Math - permite-lhe criar equações matemáticas através de uma interface gráfica ou dactilografando directamente as suas fórmulas no editor de equações.
Base - permite-lhe manipular bases de dados facilmente. Crie e modifique tabelas, formulários, consultas e relatórios a partir do Apache OpenOffice.
