Description
-----------

Apache OpenOffice® on Apache Software Foundation:in ilmainen avoimen lähdekoodin toimistosovellusperhe.

Apache OpenOffice®:ssa on kuusi toimistosovellusta: tekstinkäsittelyohjelma (ja sen internet-julkaisukomponentti), taulukkolaskentaohjelma, diaesitysohjelma, piirrosohjelma, laskentakaavaeditori ja tietokantaohjelma. Joihinkin toimintoihin vaaditaan Java Runtime Environment (JRE).

What's new in this version?
---------------------------

Security release:
Apache OpenOffice 4.1.12 on tietoturvapäivitys, joka sisältää tärkeitä bugikorjauksia. Kaikkia Apache OpenOffice 4.1.11 tai aiemman version käyttäjiä kehotetaan päivittämään nykyinen versio tähän uusimpaan versioon.

Maintenance release:
Apache OpenOffice 4.1.12 on ylläpitopäivitys, joka sisältää virheenkorjauksia ja muita parannuksia. Kaikkia Apache OpenOffice 4.1.11 tai aiemman version käyttäjiä kehotetaan päivittämään nykyinen versio tähän uusimpaan versioon.

Product features
----------------

Writer - tekstinkäsittelyohjelma, jota voit käyttää mihin tahansa kirjeistä romaanin kirjoittamiseen.
Calc - tehokas taulukkolaskentaohjelma, jossa on kaikki tarvittava datan laskentaan, analysointiin ja esittämiseen raportteina tai näyttävinä kuvaajina.
Impress - nopein ja tehokkain tapa tehdä multimediaesityksiä.
Draw - voit piirtää mitä vaan diagrammeista 3D-piirroksiin.
Math - voit luoda matemaattisia laskentakaavoja graafisella käyttöliittymällä tai suoraan kirjoittamalla kaavasi editoriin.
Base - voit luoda ja muokata tietokantoja helposti. Luo ja muokkaa tauluja, lomakkeita, kyselyitä ja raportteja suoraan Apache OpenOffice:ssa.
